## Ao entrar

Tenha certeza que está na sua branch:

```bash
git checkout sua-branch
````

Puxe atualizações do repositório principal:

```bash
git pull origin main
```

Agora você pode trabalhar no código.

---

## Ao sair

Salvar mudanças:

```bash
git add .
```

Commitar:

```bash
git commit -m "feat: o que você implementou"
```

ou

```bash
git commit -m "fix: o que você arrumou"
```

Enviar para o GitHub:

```bash
git push
```
